# نظام جباية الماء

نظام ويب بسيط لإدارة:
- المدير
- الكاتب
- المحصل
- المناطق
- الاشتراكات
- السجل السنوي لكل مشترك

## التشغيل

1. افتح التيرمنال داخل المجلد:
   - `C:/Users/lenovo/water-billing-system`
2. ثبت الحزم:
   - `npm install`
3. شغل السيرفر:
   - `npm start`
4. افتح المتصفح:
   - [http://localhost:3000](http://localhost:3000)

## الدخول الافتراضي للمدير

- الرمز: `ADMIN`
- الرقم السري: `1234`

## ملاحظات

- الكاتب يمكنه إضافة/تعديل المناطق والاشتراكات والسجلات.
- المحصل يمكنه الاطلاع فقط على الاشتراكات والسجلات المخصصة له.
- المدير ينشئ حسابات الكتاب والمحصلين ويعدل الإعدادات (أسعار الماء والتنظيف والفائدة).

## ميزات إضافية

- **تصميم رسمي + موبايل**: الواجهة صارت أجمل وتشتغل مريحة على الشاشات الصغيرة، والجداول تقدر تسحبها أفقيًا.
- **طباعة وصل PDF**: من تبويب السجل السنوي اضغط `طباعة وصل PDF` (يستخدم مكتبة jsPDF من CDN).
- **نسخ احتياطي واسترجاع**: من تبويب الاعدادات (مدير فقط):
  - `تحميل نسخة احتياطية` يحمّل ملف JSON كامل للبيانات.
  - `استرجاع نسخة` يرفع ملف JSON ويعيد كتابة البيانات.
  - بعد الاسترجاع يتم تسجيل الخروج من كل الجلسات ويلزم تسجيل الدخول من جديد.

- **استيراد Excel**: من تبويب الاشتراكات (مدير/كاتب) ارفع ملف `.xlsx` أو `.xls`:
  - إما صف عناوين يحتوي أعمدة مثل: رقم الاشتراك، اسم المشترك، اسم المنطقة، نوع الاشتراك (منزلي/تجاري)، مقياس الجباية، (هاتف اختياري).
  - أو بدون عناوين: العمود الأول رقم المشترك، الثاني الاسم، الثالث المنطقة، الرابع هاتف اختياري.
  - اختر **نظام الجباية** الافتراضي للصفوف المستوردة. وإذا وُجد عمود مقياس الجباية في الملف فسيتم استخدامه لكل صف.

- **بحث شامل**: بعد تسجيل الدخول يظهر حقل بحث أعلى الصفحة للبحث في المناطق، المشتركين (رقم/اسم/هاتف)، الموظفين (للمدير)، وأرقام وصول الدفع.
